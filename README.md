# NumberGuesser Ultimate

A simple number guessing game built in C#.

## How to Play

1. Run the application
2. Select a difficulty level (Easy, Medium, or Hard)
3. Guess the number between 1 and 100
4. Use hints if needed (limited to 3 per game)
5. Try to guess correctly before attempts run out

## Difficulty Levels

- **Easy**: 10 attempts
- **Medium**: 5 attempts
- **Hard**: 3 attempts

## Features

- Difficulty selection
- Hint system
- Game timer
- Ranking system
- Play multiple rounds

## Requirements

- .NET 10.0 or higher

## How to Run

```sh
dotnet run
```

## Build

```sh
dotnet build
```
